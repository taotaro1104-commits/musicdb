Free BGM / Sound Effect DB static site

GitHub Pages:
- Upload the contents of this folder to the Pages root, or deploy this folder with GitHub Actions.
- If you upload the folder itself under the repository root, open /site/ instead of the repository top page.

Required files:
- index.html
- styles.css
- app.js
- favicon.ico
- ogp.png
- materials-index.json
- materials.json
- materials-000.json and following chunk files
- facets.json
- material-*.html
- sitemap.xml
- robots.txt
- llms.txt
- .nojekyll

Optional for Python server:
- server.py
- musicdb.sqlite
